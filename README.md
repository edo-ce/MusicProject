# MusicProject

Web app having music artists, which can upload songs, albums...,
and listeners, which can like artists' items.

### Made With
* Python
* Flask
* SQLAlchemy
* PostgreSQL
* Jinja
* HTML/CSS/JavaScript
* Bootstrap
